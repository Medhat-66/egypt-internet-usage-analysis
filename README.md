# Egypt Internet Usage Analysis — Data Analysis

End-to-end data analysis project on internet usage patterns across Egypt.

## Key Insights
- Analyzed 6,000+ records of internet usage data across Egypt
- Identified 4 key behavioral segments across age groups and regions
- Reduced null values by 95% through data cleaning

## Steps Performed
- **Data Cleaning:** Processed and cleaned data using Python and SQL
- **EDA:** Uncovered usage trends across age groups and regions
- **Dashboard:** Built interactive Power BI dashboards to visualize KPIs
- **Reporting:** Generated analytical reports in Excel with actionable insights

## Tools & Technologies
Python, SQL, Excel, Power BI, Pandas, NumPy, Matplotlib, Seaborn
